## 9. 风险与决策点

| 风险 | 应对 |
| --- | --- |
| 一期范围扩大导致延期 | 按迭代 3→5 分批验收，每迭代独立 DoD |
| picker/datetime/upload 跨端差异 | 开发前读 `kit-ui-uts-app-compat`、官方 API 文档 |
| notification 平台限制 | 一期可先 WEB/App，小程序降级或文档说明 |
| action-sheet 与 popup 重复 | 内部复用 k-popup bottom 模式 |
| 产能分散 | 单迭代主任务不超过 6 个组件目录 |
